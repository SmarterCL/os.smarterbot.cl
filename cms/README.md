# CMS — Content Management System
# SmarterOS v4.3

Este directorio gestiona el contenido dinámico del ecosistema.

## Estructura

```
cms/
├── content/          # Contenido por tipo
│   ├── pages/        # Páginas estáticas
│   ├── snippets/     # Fragmentos reutilizables
│   └── templates/    # Templates de contenido
└── README.md
```

## Integración

El CMS se conecta con:
- **Bookish** para retrieval de contexto
- **Emdash** para generación de contenido
- **API** para servir contenido dinámico
